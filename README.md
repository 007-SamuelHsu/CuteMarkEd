## CuteMarkEd


### DESCRIPTION

A Qt-based Markdown editor with live HTML preview and syntax highlighting of markdown document.

![screenshot](https://raw.github.com/cloose/CuteMarkEd/develop/images/screenshot_01.png)

### NEWS
